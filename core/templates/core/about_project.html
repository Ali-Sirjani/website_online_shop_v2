About Project

I am Ali Sirjani, a developer in Django and Django Rest Framework, who has a passion for creating practical web applications.One of my notable projects is the "Online Store," developed using the Django framework. The Online Store is an e-commerce platform that allows users to view and purchase products on the site, even without the need to log in. Users can also save products as favorites for future reference.
In addition, users can log in using their Google accounts for a seamless shopping experience.

Additionally, I've enriched the user interface by incorporating JavaScript, coding essential features like dynamic shopping cart management, product liking functionality, and real-time product filtering based on size using AJAX. My commitment to enhancing user interaction extends to the profile section, where I've utilized JavaScript to create a seamless and secure interface for password management. Despite my initial limited knowledge of JavaScript, I took it upon myself to learn and implement these features, contributing to the overall richness of the "Online Store" project.
